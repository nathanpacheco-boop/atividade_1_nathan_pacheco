Esta é a primeira atividade utilizando HTML para criar um site de exemplo. Escolhi desenvolver um site de vendas de automóveis chamada Anger, nome formado pela inicial de cada integrante do grupo, que permanecerá o mesmo até o final do ano.
Durante a atividade, aplicamos diversas tags fundamentais para a estruturação completa de um site, trabalhando apenas com HTML, sem a utilização de CSS.
Mais adiante, será possível aprimorar o projeto, deixando-o mais profissional com a implementação de estilos utilizando CSS e também recursos em JavaScript.
